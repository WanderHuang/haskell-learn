## 基本语法

### data|type|newtype 区别

```hs
-- data    关键字 introduce一个代数数据类型。albegraic
-- type    关键字 产生synonym，一个已经existing的类型的别名，可以交换使用。(interchangeably)
-- newtype 关键字 给一个existing的类型一个独特的身份(distinct identity)。新类型和老类型不可交换的(interchangeable)
```
